ADVBOX API
 1.0.0 
OAS 3.0

Access is exclusive for clients and service providers approved by ADVBOX.

Authentication is performed through Bearer token, sent in the request header (e.g., "Authorization": "Bearer abcd1234efgh5678ijkl9012mnop3456qrst7890").

Rate limits:

  • GET: 30 requests per minute
  • POST: 100 requests per day for each route
  • PUT: 500 requests per day

⚠️ Warning: The Try it out functionality is temporarily unavailable. To test the routes, use external tools such as Postman or cURL. Payload examples are available in the documentation of each route.

Example:

  -H 'Accept: application/json' \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer your_token_here'\
